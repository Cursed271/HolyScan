
# HolyScan

A Portscanner written in Python that lists all the open ports and running services on the Host.
## Installation

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd HolyScan
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start HolyScan

```bash
  python HolyScan.py
```
